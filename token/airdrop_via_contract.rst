Block 合约空投
===================================

总量：
   5000 万 Blocks

每份数量：
   100-200 Blocks （随机）

预期用户覆盖：
   大约 30 万用户

意义：
   在 Block Network 网络效应逐渐成熟的过程中，促进知名度和共识。

期待：
   分享、传播、普及 Block Network 理念，让更多人认识区块网络。


.. NOTE
   我们检测到有组织的大量利用空投合约低成本获取 Block 的行为。
   因此，我们决定暂时中止合约空投，以避免大量的低成本 block 流入市场甚至破坏生态。
